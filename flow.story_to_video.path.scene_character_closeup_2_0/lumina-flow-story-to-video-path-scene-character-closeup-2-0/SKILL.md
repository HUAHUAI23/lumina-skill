# story_to_video path scene character closeup 2.0

## 路径定位

- 本路径面向 Seedance 2.0 / 2.0 类多模态视频模型。
- 主路径是参考图生视频，不是先把人物和场景溶成普通首帧再生视频。
- 图片负责锁定稳定事实：人物身份、服装、无人场景空间、关键道具、必要局部特写。
- 文本负责主题和运动：谁在什么场景，动作从哪里开始，镜头怎么动，情绪落到哪里。
- 提示词要短、清楚、可拍，不追求逐帧精确；过度精确会制造噪点和限制模型表演。

## 输入职责模型

- 文本：定义主题、动作方向、镜头、节奏、光线、声音和情绪。
- 人物图：锁定人物身份、脸、发型、年龄阶段、体态、服装，不承担场景。
- 场景图：锁定空间布局、时间、光线、材质和陈设，必须尽量无人。
- 特写图：锁定某一步必须看清的局部证据或转折点，例如眼神、嘴唇、手、剑身、文件袋、伤口、道具归属、情绪矛盾爆发瞬间。
- 参考图不是连续分镜播放顺序，而是同一个视频镜头的约束集合。

## 视频模式规则

- 默认优先 reference_images。
- first_frame 只在参考图生视频不可用、或确实只有一张可独立承接动作的图时使用。
- first_last_frame 极少使用；只有结尾硬结果不可拆、且不锁尾帧会造成剧情误读时才用。
- text_to_video 只用于纯空镜、抽象过场或没有任何可用参考图的低风险镜头。
- 不把“场景图 + 人物图 + 人物图”规划成普通双人同框首帧溶图。

## 分镜图规划总则

- 每个镜头先判断需要哪些参考职责，而不是先生成一张大而全的同框图。
- 标准组合是：无人物场景图 + 主要人物干净图 + 次要人物干净图 + 必要步骤特写。
- 普通对话、同框站位、轻微走位、表情变化、推镜、回头、坐下、递物，交给视频提示词语言控制。
- 只有观众必须看清、主画面自然生成不可靠，或情绪/关系矛盾需要明确转折点时，才规划特写图。
- 每张图只解决一个主要风险：身份、服装、空间、道具、动作局部、表情局部、状态证据、转折点。
- 不规划动作过程图，不把每个时间片变成一张图。

## 人物图规范

- 人物图必须是干净底图：单人、无背景或极简中性背景、无其他角色、无复杂场景陈设。
- 人物图可以通过图生图或溶图生成，用来更新服饰、发型、年龄阶段、伤势、仙侠/现代状态。
- 即使人物图需要参考场景的光线、时代或服饰，也只能输出干净人物底图，不把人物放进具体客厅、天门、街道等场景。
- 多角色镜头中，每个关键角色应有独立干净人物图，避免身份串位。
- 人物正侧背三视图只用于身份、服装和轮廓，不要求视频复刻三视图排版。
- 不用文生图生成已绑定人物的新脸；已有角色必须从绑定人物图出发做图生图或溶图。

## 场景图规范

- 场景图必须优先是无人物场景图。
- 场景图负责空间布局、景别基底、时间、光线、家具/建筑/道具陈设，不负责人物动作。
- 已有场景绑定图时，不要文生图重造场景；需要夜晚、破败、冷光、战损等状态变化时，用场景绑定图做图生图覆写。
- 场景图中如果原本有人，应在规划中要求去人化或重新生成无人物版本。
- 场景图和人物图不要溶成普通同框图；视频阶段用参考图和语言把人物放回场景。
- 场景一致性优先来自已有场景图，不能靠文字凭空生成另一个相似空间。

## 特写图规范

- 特写图只补必要步骤，不补普通过程。
- 必要特写包括：眼部反应、嘴唇吐字、捏袖口的手、擦剑反光、文件袋、玄天玉、伤口、血迹、符纸、门牌、道具归属、状态材质、情绪矛盾突出时的转折点。
- 转折点特写用于关系或情绪发生不可逆变化的瞬间，例如玩笑卡住、眼神从回避落向对方、手指停止动作、呼吸停顿、笑意消失、恐惧落下。
- 转折点特写必须对应一个具体可见局部，不写成抽象“悲伤特写”或“震撼氛围图”。
- 特写图可以用人物图、道具图、场景图做图生图或溶图，但输出必须仍是单一局部特写。
- 特写图不同时承担人物全身、场景全貌和多个道具。
- 道具或部位特写需要场景一致性时，可以参考场景图的光线和材质，但不要把整个场景塞进特写。
- 准确中文文字不作为生成成功条件；文件、纸张、牌匾只生成材质、留白、模糊痕迹或可后期贴字区域。

## 溶图规则

- 允许溶图生成干净人物图：例如用人物绑定图 + 服装/状态参考，输出无场景的当前阶段人物底图。
- 允许溶图生成必要特写：例如人物手部 + 道具 + 场景光线，输出手握道具特写。
- 允许用场景绑定图做图生图生成无人场景状态图。
- 禁止溶图生成普通双人中景、普通同框首帧、普通站位建立、普通对话首帧。
- 禁止把人物图和场景图溶成“一张看起来完整”的主图后再视频生成；这会让参考图路径退化为旧的首帧路径。
- 有可用绑定图时不要用文生图替代；文生图只在完全缺失且不断镜必须兜底时使用。

## 图片数量和顺序

- Seedance 2.0 支持最多 9 张图片；本路径常规使用 2-6 张，复杂镜头可到 7-9 张。
- 图数由职责决定，不为了凑数，也不为了少图丢掉关键人物、关键证据或关键转折点。
- 推荐顺序：无人物场景图在前，主要人物干净图随后，次要人物干净图再后，关键道具或步骤特写最后。
- 每张图片说明只写一个短职责，例如“图片1锁定无人客厅冷光空间”，“图片2锁定林照月白衣女修身份”，“图片4锁定文件袋特写”。
- 不使用“上图、图1、参考图1、Reference image 1”等混乱称呼；统一使用“图片1、图片2、图片3”。

## 分镜规划工作法

1. 先判断镜头主题和运动：谁、在哪里、动作从哪里开始、镜头怎么动、情绪落到哪里。
2. 再列参考职责：需要哪些干净人物图、哪张无人物场景图、哪些必要特写。
3. 对每个已绑定人物，优先复用或图生图生成干净人物底图。
4. 对每个已绑定场景，优先复用或图生图生成无人物场景图。
5. 只在关键证据看不清，或情绪/关系转折必须被观众看见时补特写图。
6. 选择 reference_images；除非能力不可用或镜头极简单，才退到 first_frame。
7. 最后检查是否误生成了普通同框溶图；如果是，改回人物图 + 无人物场景图 + 必要特写。

## 视频提示词写法

- 用“固定参数 + 时间段正文”的短格式，不输出冗长栏目。
- 固定参数只保留必要项，例如画风、时长、比例。
- 如果输入是原始剧本、对白、旁白或分场文稿，先提取固定参数，再改写成镜头化时间段，不要原样堆剧情说明。
- 正文优先按“时间 + 景别/运镜 + 主体 + 动作 + 场景/光线 + 台词/声音”的顺序组织。
- 9 秒以上默认拆成 2-4 个时间段；13-15 秒可以拆成 3-5 个时间段。时间段用于控制镜头重心，不对应新增图片。
- 正文按时间写镜头，格式接近：

```text
画风: 3D仙侠

0.0-4.0秒。 时间: 夜晚。 [全景/固定]深夜客厅，窗外可见城市灯光。客厅空无一人，环境安静。旁白（VO）: 「……」
4.0-6.0秒。 时间: 夜晚。 [全景/推镜]镜头从门口向客厅内推进。客厅中央的沙发上坐着黑袍男人。
6.0-7.0秒。 时间: 夜晚。 [特写/固定]黑袍男人放在膝盖上的手。手指惨白，指甲异常修长。
```

- 如果是连续对话，也可以写成自然镜头句：

```text
镜头从三人同框的中景开始，中间男子开口说道：“We need to clear this up.” 接着切至左侧女子近景，冷静回应：“I’ve already made my choice.” 最后镜头切回三人同框中景，气氛变紧。
```

- 对短镜头可用更紧凑格式：

```text
画风: 3D仙侠

0.0-1.0秒。 时间: 清晨。 眼部特写固定镜头。林照月猛然睁开眼，眼神冰冷地看向画外。
1.0-3.0秒。 时间: 清晨。 嘴唇特写固定镜头。林照月嘴唇微动，冷冷吐字。林照月: 「加。」
```

## 视频提示词约束

- 核心写主题 + 运动，不写长篇散文化解释。
- 一段镜头只指定一种主运镜：固定、缓慢推近、轻微跟拍、平稳横移等。
- 动作写方向和过渡，不写逐帧姿态；优先写缓慢、连续、可衔接的小动作。
- 情绪写可见身体信号，例如眼神回避、嘴角僵住、呼吸停顿、肩背不动，不只写“悲伤、愤怒、震惊”。
- 台词只放在对应时间段的关键位置，不把整段画面写成念白。
- 对白、VO、OS 保留为台词信息，不生成字幕或画面文字。
- 声音可短写环境音、动作音和 BGM 情绪；不要把声音扩写成大段说明。
- 结尾稳定性约束保持短：人物身份稳定、动作自然、画面无闪烁、无字幕水印。

## Seedance 方法论落地

- 固定参数和正文分离：画风、时长、比例放开头，镜头内容放时间段正文。
- 时间轴优先：不要输出一整段散文化视频描述；至少给出起止时间或明确的镜头顺序。
- 主公式是：时间 + 镜头/运镜 + 主体 + 动作 + 场景/光线 + 声音/对白 + 短约束。
- 镜头语言克制：一个时间段一个主景别或主运镜，不同时堆推、拉、摇、移。
- 动作可拍：用停顿、抬眼、侧头、呼吸、手指停住、物品滑落等小动作表达情绪转折。
- 参考图职责明确：每张图只锁身份、空间、道具、特写或转折点，不把参考图当作逐帧动作。
- 约束词简短：只保留人物身份稳定、面部稳定、动作自然、无字幕水印等高收益约束。

## 质量标准

- 规划结果能看出：人物图是干净底图，场景图无人，特写图对应具体步骤、证据或转折点。
- 普通同框关系不依赖溶图，靠参考图 + 语言控制完成。
- 有绑定人物/场景时，不用文生图重造身份或空间。
- 视频提示词短、清楚、有时间和镜头，不再输出“画面与动作 / 镜头要求 / 台词与旁白 / 声音设计”这类冗长包。
- 每张图职责明确，不是海报、拼贴、过程帧或完整分镜组。
- 首尾帧极少出现；大多数镜头由 reference_images 或必要时 first_frame 承接。

## 失败回退

- 画面漂移：优先补干净人物图或无人物场景图，不补普通同框溶图。
- 人物串位：为每个关键角色保留独立干净人物图，并在视频提示中写清站位关系。
- 场景不一致：回到已有场景图做无人物图生图，不用文生图造新空间。
- 局部证据或情绪转折看不清：只补一张局部特写，不扩成多张过程图。
- 提示词过长：删掉装饰、心理解释、重复形容词，只保留时间、景别/运镜、主体、动作、场景、声音。
