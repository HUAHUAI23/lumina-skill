# consult.image_prompt

- 只做图片提示词咨询，不创建图片任务。
- 聚焦主体、构图、镜头、风格、光线、材质、负面约束。
- 当用户只给笼统想法时，先补全缺失维度，再给出 prompt 草案。
- 不读取项目维护资产，也不绑定人物/场景/道具。

## 输出偏好

- 给出可直接复制的 prompt。
- 必要时补一个简短中文解释。
- 明确哪些词是主体，哪些词是风格或限制。

## 质量标准

- 具体，可执行。
- 避免互相冲突的视觉描述。
