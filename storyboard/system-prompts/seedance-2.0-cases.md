# Seedance 2.0 案例模板库（System Prompt Pack）

本文件用于系统提示词一次性注入，不放在 `resources/`。

## 0. 类型索引
- C01 多模态新建
- C02 一致性强化
- C03 动作/运镜复刻
- C04 一镜到底
- C05 视频延长
- C06 定向编辑
- C07 特效复刻
- C08 音频/音色驱动
- C09 音乐卡点
- C10 分镜驱动补全
- C11 商业广告成片

## 1. 共用模板头
```text
[入口]
- 首尾帧 / 全能参考

[素材映射]
- @图片1: [首帧/角色参考/场景参考]
- @视频1: [动作/运镜/节奏/待延长/待编辑]
- @音频1: [BGM/音色/节奏]

[目标]
- 类型: [C01-C11]
- 生成时长: [4-15s]
- 画幅: [16:9/9:16/1:1]

[硬约束]
- 一致性要求: [...]
- 禁止项: [...]
- 是否切镜: [可切镜/一镜到底]
```

## 2. 关键模板

### C04 一镜到底
```text
以 @图片1 为首帧，不要切镜头。
镜头从 [A] 跟随主体到 [B]，再到 [C] 收束。
保持方向连续，不反轴。
```

### C05 视频延长
```text
将 @视频1 延长 [N] 秒。
新增部分按 [0-2s][2-4s][4-Ns] 拆分描述。
保持原片光影、色调、运动方向一致。
```

### C06 定向编辑
```text
基于 @视频1，仅修改 [X-Y秒]：将 [原动作] 改为 [新动作]。
其余时段保持不变。
```

### C11 商业广告成片
```text
以 @图片1 产品为主体，
按 [0-3s/3-8s/8-12s/12-15s]：
Hook -> 卖点证据 -> 使用场景 -> Logo+CTA。
```

## 3. 团队快捷模板

### Q01 电商产品 15s
```text
0-3s 建立品牌与产品；3-8s 两个卖点特写；
8-12s 使用场景；12-15s Logo + CTA。
```

### Q02 口播种草 10s
```text
0-2s 痛点；2-7s 解决方案；7-10s CTA。
```

### Q03 一镜到底追逐 8-12s
```text
不切镜头，路径连续，终点明确。
```

### Q04 剧情短片 15s
```text
建立 -> 冲突 -> 收束，三段推进。
```

### Q05 MV 卡点 12s
```text
按强拍做景别/动作/转场变化。
```

### Q06 视频延长 6s
```text
2s+2s+2s 推进，风格连续。
```

### Q07 局部改写 3-6s
```text
仅改指定区间，其余不变。
```

## 4. 超限回退
```text
若输入超限：
1) 保留优先素材（角色一致性 > 动作 > 运镜 > 音频）
2) 拆分 A/B 段生成
3) 第二段引用第一段结果
```
