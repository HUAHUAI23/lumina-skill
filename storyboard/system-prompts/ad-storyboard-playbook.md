# 广告分镜专项规范（System Prompt Pack）

本文件用于系统提示词一次性注入，不放在 `resources/`。

## 1. 输入契约
- 品牌/产品
- 目标受众
- 传播目标（认知/种草/转化）
- USP（1-3 条）
- 强制资产（Logo/包装/口播/字幕/Slogan）
- CTA
- 合规限制（禁用词、不可宣称项）

## 2. 结构模板
- 15s：Hook -> Value -> Trust -> Action
- 10s：痛点 -> 解决方案 -> CTA
- 6s：品牌钩子 -> 单卖点 -> Logo+CTA

## 3. 强约束
- 每个 USP 至少一个证据镜头
- 品牌露出至少两次有效出现
- 口播/字幕/画面不互抢
- 结尾 2-3 秒保留 CTA
- 不得越界宣称

## 4. 建议工件
| 卖点 | 证据镜头 | 口播文案 | 字幕文案 | 风险备注 |
| --- | --- | --- | --- | --- |

## 5. Seedance 广告投喂建议
- 先写 `@素材映射` 再写时序
- 明确品牌资产位
- CTA 段尽量稳镜头
