# task.prop_extract.image_prompt

## 适用场景

- 已完成道具抽取后，为单个道具整理一条可直接出图的任务提示。

## 负责什么

- 把道具名称、类别和关键识别特征压实成一条可执行图片任务。
- 让道具形状、材质、颜色、纹理、发光效果和比例关系稳定。
- 需要时补充克制的反向约束，减少人物、手部或复杂背景干扰。

## 不负责什么

- 不新增道具。
- 不把一个道具拆成多条任务。
- 不改写上游已经确认的道具身份。

## 判断原则

- 主体必须稳定可辨认，画面只围绕一个核心道具。
- 默认生成干净的道具参考图，避免人物手持、复杂场景和遮挡。
- 对灵物、法器等可加入克制的光效，但不要掩盖本体形状。
- 提示内容优先服务道具设定建立，而不是剧情分镜。

## 工作方法

- 先吃透当前道具最该被看见的轮廓、材质和辨识点。
- 再决定适合的构图、背景简洁度和光线。
- 最终整理成一条单图任务，适合沉淀为道具参考图。

## 质量标准

- 一眼能看出是什么道具。
- 画面目标明确，没有空泛修辞。
- 结果适合后续作为多参考视频中的道具参考。

## 失败回退

- 信息不够时，优先保住名称、轮廓和核心材质。
- 不额外捏造原文没有支持的机关、铭文或配件。
