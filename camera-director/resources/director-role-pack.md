# 导演角色包（Director Role Pack）

本文件整合分镜导演、运镜专家、动画分镜优化导演和提示词工程角色设定，用于模式 B/B-Review 的知识参考。

---

## 1. 专业影视分镜师 / 导演

### 任务目标
- 把小说/剧本转为可拍摄、可生成、可执行的分镜脚本
- 核心是视觉化翻译，而非摘要

### 核心流程
1. 通读定调（题材 + 情绪基调）
2. 场次划分（Scene）
3. 镜头拆解（Shot）
4. 视听填充（景别/运镜/画面/音效）

### 强约束
- 心理描写必须视觉化（微表情、动作、OS、象征画面）
- 动作戏偏短镜，情感戏偏长镜
- 优先用 Markdown 表格输出

## 2. AI 运镜大师（Cinematic Camera Specialist）

### 物理逻辑硬规则
- `Pan/Tilt`：机位固定，镜头转动
- `Dolly/Truck/Crane`：机位移动
- 禁止把互斥运动写在同一时刻

### 高价值技法速查

| 技法 | 英文 | 适用效果 |
| --- | --- | --- |
| 希区柯克变焦 | Dolly Zoom / Vertigo Effect | 主体大小近似不变，背景压缩/扩张，制造眩晕 |
| 环绕弧形运镜 | Orbit / Arc Shot | 强化空间感、史诗感、主体展示 |
| 第一人称穿越 | FPV Drone Fly-through | 沉浸穿梭、速度感、空间纵深 |
| 主观视角 | POV Shot | 角色眼睛视角，增强代入 |
| 荷兰角推进 | Dutch Angle Push In | 失衡+压迫，强化心理崩塌 |
| 子弹时间 | Bullet Time | 时间冻结感，摄影机高速绕行 |
| 跟焦/移焦 | Rack Focus / Pull Focus | 焦点转移，引导观众注意力 |
| 甩镜头 | Whip Pan / Swish Pan | 动态转场、节奏跳跃 |

## 3. 动画导演分镜优化（B-Review）

### 目标
在不推翻剧情、人设、叙事目标前提下，优化镜头表达与制作可行性。

### 重点
- 叙事与表演并重
- 兼顾原画工作量、中期难度、周期和成本
- 对 TV 动画/剧场版/短视频采取不同节奏标准

### 交付结构
1. 导演全局评估
2. 逐镜修改建议
3. 镜头组接与节奏优化
4. 视听与表演整体升级
5. 制作落地全流程提示

## 4. 人物微表情导演参考

| 目标情绪 | 面部微表情（Face） | 肢体/道具动作（Action） |
| --- | --- | --- |
| 害羞/心动 | 咬下唇、眼神躲闪、睫毛颤动 | 低头、别头发、用物件遮脸 |
| 暗自窃喜 | 嘴角上扬但克制、眯眼 | 手背捂嘴、肩膀轻耸、偷看手机 |
| 感激/释然 | 闭眼深呼吸、眉头舒展、眼角湿润 | 长呼气、双手合十、身体后靠 |
| 自信/得意 | 抬下巴、挑眉、凝视目标 | 整理衣领、转笔、抖肩落灰 |
| 悲伤/委屈 | 下巴颤抖、眼眶泛红、望天压泪 | 抓衣角、动作机械化 |
| 隐忍愤怒 | 咬肌紧绷、鼻翼扩张、目光变冷 | 拳头发白、折断笔、指甲掐掌心 |
| 不耐烦/嫌弃 | 皱眉、翻白眼、嘴角下撇 | 看表、叹气、指尖敲击 |
| 绝望/崩溃 | 眼神失焦、嘴角抽动、木然 | 抱头、抓发、靠墙滑落 |
| 恐惧/紧张 | 瞳孔颤动、额头冷汗、喉结滚动 | 抠指甲、疯狂按笔、频繁吞咽 |
| 撒谎/心虚 | 频繁眨眼、不敢对视、舔嘴唇 | 摸鼻子、拉衣领、手挡嘴 |
| 震惊/呆滞 | 嘴微张、瞳孔放大、动作定格 | 手中物掉落、身体僵住 |
| 深思/算计 | 眼睛微眯、注视一点 | 摩挲戒指、把玩打火机、摸下巴 |

## 5. 动作/战斗词库（可直接用于 Prompt）

### 5.1 攻击
- heavy punch, roundhouse kick, jump kick
- slash, thrust, downward strike
- rapid combo, ground slam

### 5.2 防御与反应
- block/parry, deflect, dodge
- backflip evade, stagger backward
- knockback, recoil

### 5.3 武器与魔法
- draw blade / iaido, spinning sword flourish
- charge energy, cast fireball, summon shield
- bow draw + arrow release, hand seals / chant
