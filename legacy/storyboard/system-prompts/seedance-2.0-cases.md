# Seedance 2.0 案例模板库

本文件用于系统提示词一次性注入，不放在 `resources/`。

## 0. 类型索引
- C01 多模态新建
- C02 一致性强化
- C03 动作/运镜复刻
- C04 一镜到底
- C05 视频延长
- C06 定向编辑
- C07 特效复刻
- C08 音频/音色驱动
- C09 音乐卡点
- C10 分镜驱动补全
- C11 商业广告成片

## 1. 共用模板头
```text
[入口]
- 首尾帧 / 全能参考

[素材映射]
- @图片1: [首帧/角色参考/场景参考]
- @视频1: [动作/运镜/节奏/待延长/待编辑]
- @音频1: [BGM/音色/节奏]

[目标]
- 类型：[C01-C11]
- 生成时长：[4-15 秒]
- 画幅：[16:9/9:16/1:1]

[硬约束]
- 一致性要求：[...]
- 禁止项：[...]
- 是否切镜：[可切镜/一镜到底]
```

## 2. 关键模板

### C04 一镜到底
```text
以 @图片1 为首帧，不要切镜头。
镜头从 [A] 跟随主体到 [B]，再到 [C] 收束。
保持方向连续，不反轴。
主体外观、服装和主色调保持一致。
```

### C05 视频延长
```text
将 @视频1 延长 [N] 秒。
新增部分按 [0-2 秒][2-4 秒][4-N 秒] 拆分描述。
保持原片光影、色调、运动方向一致。
```

### C06 定向编辑
```text
基于 @视频1，仅修改 [X-Y 秒]：将 [原动作] 改为 [新动作]。
其余时段保持不变。
不得改动主体身份、整体时长和未指定区域。
```

### C11 商业广告成片
```text
以 @图片1 产品为主体，
按 [0-3 秒 / 3-8 秒 / 8-12 秒 / 12-15 秒]：
开场钩子 -> 卖点证据 -> 使用场景 -> 品牌露出与行动引导。
```

## 3. 团队快捷模板

### Q01 电商产品 15s
```text
0-3 秒建立品牌与产品；
3-8 秒展示两个卖点特写；
8-12 秒给出使用场景；
12-15 秒完成品牌露出与行动引导。
```

### Q02 口播种草 10s
```text
0-2 秒抛出痛点；
2-7 秒给出解决方案；
7-10 秒收束到行动引导。
```

### Q03 一镜到底追逐 8-12s
```text
不切镜头，路径连续，终点明确。
主体速度变化要自然，不突然瞬移。
```

### Q04 剧情短片 15s
```text
建立 -> 冲突 -> 收束，三段推进。
每一段都要有明确景别与动作变化。
```

### Q05 MV 卡点 12s
```text
按强拍做景别、动作和转场变化。
优先保证节奏同步，再做风格化修饰。
```

### Q06 视频延长 6s
```text
2 秒 + 2 秒 + 2 秒推进，风格连续。
```

### Q07 局部改写 3-6s
```text
仅改指定区间，其余不变。
修改段落必须与前后运动方向和光影连续。
```

## 4. 超限回退
```text
若输入超限：
1) 保留优先素材（角色一致性 > 动作 > 运镜 > 音频）
2) 拆分 A/B 段生成
3) 第二段引用第一段结果
```

## 5. 中文成片样例

### 广告成片样例
```text
[入口]
- 全能参考

[素材映射]
- @图片1：产品主体与品牌包装参考
- @视频1：水雾运动与镜头节奏参考

[目标]
- 类型：C11
- 生成时长：15 秒
- 画幅：9:16

[硬约束]
- 产品包装形态不得变化
- 品牌色保持浅蓝与白色
- 禁止额外人物与杂乱背景
- 可切镜

[时序]
- 0-3 秒：用强高光和水雾建立产品记忆点
- 3-8 秒：连续两个卖点特写，展示喷雾细腻与肤感轻薄
- 8-12 秒：展示用户在户外使用的清爽状态
- 12-15 秒：品牌露出、口播收束、明确行动引导
```
