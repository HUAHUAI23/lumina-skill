# Image Prompt Library（预加载）

本文件用于 `image_prompt` skill 的默认预加载，不依赖工具读取。内容聚焦“高频可复用词库 + 中文示例 + 平台化控制要点”。

---

## A. 模型与平台速查

| 版本/平台 | 适合场景 | 优先策略 |
|---|---|---|
| `5.0 Lite` | 时效知识图、复杂语义图 | 先明确目标与信息点 |
| `4.5` | 商业写实、人像、产品精修 | 先出构图稿，再做精修 |
| `4.0` | 批量草图、探索方向 | 控制成本，快速试版 |
| `3.1` | 插画、概念、艺术风格 | 主风格单一明确 |
| `3.0` | 文字海报、排版图 | 文字内容单独写死 |
| OpenAI / GPT Image | 多轮编辑、连续修图 | 用上一轮结果继续编辑 |
| Runway Gen-4 | 自然语言整句、参考图一致性 | 用完整句、少用负面词 |
| Imagen | 从短提示迭代到长提示 | 逐轮加细节 |
| Midjourney | 风格探索、样式控制 | 先简单，再加风格码 |

## E. 强约束词库

### 通用硬约束
- 主体不可变
- 构图稳定
- 不新增人物
- 不新增无关道具
- 无水印
- 无界面文字
- 无编号
- 无字幕条
- 无 HUD

### 角色设定硬约束
- 全身
- 自然站立
- 正面视角
- 无剧情动作
- 背景极简
- 轮廓清晰

### 场景空镜硬约束
- 绝对无人
- 不出现主角
- 空间层次明确
- 远中近景关系清晰

### 广告图硬约束
- 产品形态准确
- 品牌色准确
- 留白区明确
- 卖点可视化
- 包装不变形

## F. 平台化写法提醒

### OpenAI / GPT Image
- 适合连续编辑与局部修正
- 同一项目建议沿用上一轮图继续修改
- 文本要求明确到“保留什么、修改什么、不要改什么”

### Runway Gen-4
- 自然语言整句更稳
- 适合“主体 + 场景 + 构图 + 光影 + 风格”的顺序
- 参考图更适合锁环境和角色一致性

### Imagen
- 先用短提示快速验证方向
- 再逐轮补细节、上下文与风格
- 当要求人物脸部更精细时，要显式强调“肖像、面部细节”

### Midjourney
- 想更贴近提示，就降低风格自由度
- 风格码只控制风格，不替代主体描述
- 复杂风格探索时，先用简单提示验证样式，再叠加参数

## G. 中文示例

### G1 文生图
```text
请生成一张竖版人物海报。主体是一位成年亚洲女性，短发，黑色短风衣与高筒靴。场景是夜雨中的霓虹街道，地面有清晰倒影和薄雾。画面采用中景平视，三分法构图，强调冷暖对比光和湿润空气感，突出面部轮廓、皮革材质和雨水颗粒。必须保留短发、黑风衣和高筒靴，不要出现第二人物、字幕、水印和界面元素，画幅为9:16。
```

### G2 图生图
```text
以参考图为唯一主体依据，在不改变人物身份、发型、服装和主体朝向的前提下，把画面提升为更干净的商业质感。保留原有构图关系与主要光位，强化面部细节、衣料纹理和雨夜反光，清除画面上的字幕、水印、角标和界面元素，输出自然、不塑料、不拼接的高清结果。
```

### G3 叙事宫格
```text
序列头：风格为电影写实；题材为都市追逐；画幅为1:1；总格数为9；角色与场景必须连续一致。

第1行（铺垫）
格1：远景，夜雨街道建立环境，角色从画面左侧进入，霓虹反光明显。
格2：中景，角色回头，肩颈紧绷，察觉身后异动。
格3：近景，眼神坚定，雨滴贴着脸颊滑落，背景灯牌虚化。

参数：连续性优先，禁止画面文字、编号、水印和 HUD。
```

### G4 广告 KV
```text
请生成一张防晒喷雾广告主视觉。主体为半透明磨砂瓶身，品牌色为浅蓝与白色。场景是清爽水雾环境，柔和日光从侧后方打入，瓶身边缘有干净高光。画面居中偏左，右上预留标题区，底部预留 CTA 区。必须突出清透、防晒、轻薄不黏腻的卖点，禁止错误包装、夸张液体飞溅、杂乱背景和水印，画幅为16:9。
```

## H. 常用负面词

仅在目标平台支持且确有必要时追加：

```text
低清晰度，结构畸形，多余手指，多余肢体，脸部重复，文字污染，水印，角标，字幕条，HUD，曝光过度，颜色脏乱
```
